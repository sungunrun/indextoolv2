In perception, each referential application is an event, or aspect of an occurrent state, at
a time in the psychological system of an individual. Referential applications are not ability­
general. That is, they cannot be freely exercised again and again—unlike a capacity to perceive
something as square. Referential applications are ability­ particular. They are
individuated by occurrent events, not by freely repeatable competencies. They are, in that
sense, occurrence­ based.

I say that referential applications are occurrence-based, rather than flatly occurrent,
because, as noted, two occurrent events can instantiate the same referential application.
Then, the same referential application is a type, not an occurrent event. Referential applications
in perception can be preserved in perceptual memory. Then, the representational
identity of the referential application in memory is constitutively fixed by the representational
identity of the referential application in perception that it preserves. For example, a
referential application may be fixed by an occurrent perceptual event. The application in
memory is another, later event. But it is an instance of the same referential application that
occurred in the perception. The referential application is occurrence-based in that its
identity depends on the initial occurrence in perception. Further events can instantiate
the same referential application in perceptual memory, if and only if they are ana phor ical
ly connected to the original referential application event in perception. That is, the referential
representation by the second event depends entirely on that of the first event, and
the causal referential route to any referent for the second event goes through the route of
the first.

A similar anaphoric phenomenon occurs in linguistic discourse. One person can pick
out an object demonstratively. A recipient of communication from that person can take up
the demonstrative pronoun in such a way is to intentionally connect to the original occurrence.
Then the recipient’s demonstrative pronoun does not initiate a new referential
application. It utilizes the sender’s referential application as the antecedent of the recipient’s
demonstrative pronoun. The recipient’s demonstrative application event relies essentially
on the sender’s for its referring. (And such anaphora, both in perception and in
discourse, can occur even if the referential applications lack a referent.) In all anaphoric
cases, the sender’s and recipient’s application events count as a single demonstrative application.
In being tied to an initial application event anaphorically or pronominally, the
recipient’s application event has the same semantical function and semantical power. That
fact grounds taking the two events as instances of one referential application.
